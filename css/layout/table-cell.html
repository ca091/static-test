display: table-cell; vertical-align: middle;垂直居中

元素的垂直居中对齐，关联伸缩等
display:table-cell的元素对宽度高度敏感，对margin值无反应，响应padding属性
table-cell同样会被其他一些CSS属性破坏，例如float, position:absolute;
应用:两栏自适应 和 等高布局
